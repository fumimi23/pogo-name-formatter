# pogo-name-formatter

ポケモンGOの個体値を入力すると、ポケモンに付けるニックネームを生成するツール。

**公開URL: https://fumimi23.github.io/pogo-name-formatter/**

スマホのブラウザで上記URLを開けばすぐに使える（GitHub Pages で公開）。

## 名前の形式

```
個体値の割合(小数第1位) 半角スペース 個体値16進数(攻撃→防御→HP)
```

- **個体値の割合**: `(攻撃 + 防御 + HP) / 45 * 100` を小数第1位で表示
  - ただし `100.0`(15/15/15) のときのみ、文字数節約のため `99.9` と表示する
- **個体値16進数**: 攻撃・防御・HP の各値(0〜15)を1桁の16進数(0〜F)で連結

### 例

| 攻撃 | 防御 | HP | 割合 | 16進数 | 名前 |
|----:|----:|---:|------|--------|------|
| 15 | 15 | 15 | 100.0→99.9 | FFF | `99.9 FFF` |
| 15 | 14 | 13 | 93.3 | FED | `93.3 FED` |
| 0 | 0 | 0 | 0.0 | 000 | `0.0 000` |

## 入力

攻撃 / 防御 / HP をそれぞれ 0〜15 で指定する。
（入力方式の参考: みんポケ「新・個体値ランクチェッカー」 https://9db.jp/pokemongo/data/6606 ）

## 動作環境

インストール不要の単体 HTML ファイル。ブラウザで開くだけで動作し、スマホでもオフラインで利用できる。

GitHub Pages（`main` ブランチのルート）で公開している。`index.html` を更新して `main` に取り込めば、自動的にサイトへ反映される。
